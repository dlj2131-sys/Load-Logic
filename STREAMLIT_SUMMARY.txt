QUICK START:

1. Install dependencies:
   pip install streamlit requests pandas

2. Terminal 1 - Start FastAPI backend:
   python -m uvicorn app.main:app --reload

3. Terminal 2 - Start Streamlit:
   streamlit run streamlit_app.py

4. Open browser:
   http://localhost:8501

DEPLOYMENT OPTIONS:

RECOMMENDED: Streamlit Cloud + Render

1. Backend on Render ($7/month)
   - FastAPI API running
   - Auto-deploys from GitHub
   
2. Frontend on Streamlit Cloud (Free)
   - Streamlit app hosted
   - Set API_BASE_URL environment variable
   
3. Total Cost: ~$7/month

Alternative: Docker Compose
- Run both locally or on any server
- Full control, self-hosted

TROUBLESHOOTING:

Q: "API server is not responding"
A: Make sure backend is running and API_BASE_URL is correct

Q: "Could not geocode depot"
A: Use coordinates (lat,lon) instead of address, or set GOOGLE_MAPS_API_KEY

Q: "ModuleNotFoundError"
A: pip install -r streamlit_requirements.txt

Q: Routes aren't being created
A: Check backend logs for errors, verify API is healthy
