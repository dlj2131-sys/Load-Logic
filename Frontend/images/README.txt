IMAGE ASSETS NEEDED:
====================

Please add the following images to this folder:

1. logo.png (or logo.jpg)
   - Company logo for the header
   - Recommended size: 200x60px or similar
   - Transparent background preferred

2. hero-background.jpg (or hero-image.jpg)
   - Hero section background image
   - Recommended size: 1920x1080px or larger
   - Should show heating oil truck, delivery, or related imagery

3. feature-affordable.jpg
   - Image for affordable feature card
   - Recommended size: 400x300px
   - Could show pricing, money, or value-related imagery

4. feature-dependable.jpg
   - Image for dependable feature card
   - Recommended size: 400x300px
   - Could show reliability, trust, or service-related imagery

5. feature-quick.jpg
   - Image for quick feature card
   - Recommended size: 400x300px
   - Could show speed, delivery truck, or fast service imagery

6. about-image.jpg
   - Image for about section
   - Recommended size: 800x600px
   - Could show company, team, or service area imagery

7. truck-image.jpg (optional)
   - Delivery truck image
   - Can be used in various sections
   - Recommended size: 1200x800px

Note: All images should be optimized for web (compressed) to ensure fast loading times.
